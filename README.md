# Nuance Audio

A professional offline audio editor built as a single HTML file. No server, no install, no internet required after loading.

## Features
- 11 real-time effects: EQ, Compressor, Distortion, Bit Crusher, Chorus, Delay, Reverb, Tremolo, Noise Gate, Stereo Pan, Pitch Shift
- Waveform editor with trim, delete, fade, amplify, normalize, silence
- Mic recording directly into the editor
- Live FX preview — hear changes as you drag sliders
- Spectrum analyzer, clip detection, zoom view
- Export as 16-bit PCM WAV
- Works fully offline — Web Audio API only
